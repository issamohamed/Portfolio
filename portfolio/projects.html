Connect 4

November 24, 2021

As an avid lover of classic games, for a class project, I embarked on a thrilling project to create my own rendition of "Connect Four" in Python. To bring this game to life, I harnessed the power of the Tkinter module, a popular graphical user interface library, and augmented it with my custom graphics library, aptly named "pgl". Through careful design and meticulous coding, I managed to strike a good balance between aesthetics and functionality, resulting in an immersive and captivating gaming experience. With a few deft clicks of the mouse, players can strategically place their moves on a sleek and graphical game board, with each move triggering a cascade of events as the game state is dynamically updated. To provide an engaging challenge, I implemented a formidable bot opponent armed with a repertoire of clever strategies. This AI adversary carefully evaluates the game state and executes cunning maneuvers to thwart the player's paths to victory. Additionally, behind the scenes, my code quickly tracks every move, cataloging player choices, bot counteractions, and the remaining open spots on the board. Leveraging this wealth of information, I was able to painstakingly engineer an intelligent win-condition checker, capable of swiftly identifying when a player triumphs over their opponent or when the game ends in a draw. Overall, this project provided a great learning experience regarding how to tackle dynamic and conditional event handling.

Sudoku Solver

December 3, 2022

Growing up, I never fared well against Sudoku Puzzles. Inspired by this shortcoming, I began the development of “The Sudoku Solver”, a small project that uses a backtracking algorithm to solve Sudoku puzzles. By systematically exploring number combinations for each empty cell and adhering to Sudoku rules, the solver efficiently finds the optimal solution. It starts by identifying empty cells and tries numbers from 1 to 9, ensuring their validity in rows, columns, and 3x3 grids. If a number is invalid, it backtracks and explores alternative options. This process continues until the puzzle is solved or deemed unsolvable. The Sudoku Solver is a valuable tool for Sudoku enthusiasts and developers interested in algorithmic problem-solving, offering an enjoyable and efficient way to conquer Sudoku challenges.

ATM Machine

December 24, 2022

This project is a user-friendly Java application that emulates the functionality of an Automated Teller Machine (ATM). With this program, one can effortlessly check their account balance, withdraw funds, deposit funds, and exit the system. To use this specific program, one must compile with the command “javac ATM.java” before running, ensuring they have the Java Development Kit (JDK) installed. Upon launching the program, a menu is displayed, presenting options for checking one’s account balance, withdrawing funds, depositing funds, and exiting. By selecting option 1, one can quickly check their account balance, which will be displayed on the screen. Additionally, option 2 allows one to withdraw funds by entering the desired withdrawal amount. The program verifies that the amount is within one’s available balance and provides a success message upon a valid withdrawal. In case of insufficient funds, an appropriate error message is shown. For depositing funds, one can choose option 3 and input the desired deposit amount. Accordingly, the program in turn, adds the deposited amount to your current balance and displays a success message. Lastly, if the user selects option 4, the system exits gracefully, displaying a final farewell message. Overall, this project was a great opportunity for me to get familiar with and practice try catching and exception handling.

Stock Price Fetcher

January 8, 2023

This Java program allows users to retrieve the current stock price of any company listed on the New York Stock Exchange (NYSE). It's a command-line application that interacts with the Alpha Vantage API to fetch real-time stock data.The program offers a user-friendly interface where users can enter the NYSE abbreviation of the desired stock. It validates the input, retrieves the stock price using the API, and displays it on the screen. Users can continue looking up stock prices for multiple stocks, with the option to exit the program at any time. During the development process, I faced a few challenges. One of the key challenges was integrating with the Alpha Vantage API and handling the JSON response returned by the API. I used the org.json library to parse the JSON data and extract the necessary information, such as the stock symbol and price. Another challenge was ensuring a smooth user experience and handling user input validation. I implemented error handling to handle cases where users input an invalid stock symbol or provide incorrect responses when prompted. Overall, the Stock Price Fetcher program provides a simple yet powerful tool for retrieving real-time stock prices.
